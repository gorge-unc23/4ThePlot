# EventApp: User Scenarios List

## Goer/Participant Scenarios

| ID | Scenario Name | Scenario Description |
| :--- | :--- | :--- |
| GO-01 | Follow Preferred Categories | A goer interested in music and fitness wants to follow those categories to get relevant event recommendations. |
| GO-02 | Check Host Credibility | Before joining a paid event, a goer wants to read host ratings and reviews to make a safer choice. |
| GO-03 | Enter Waitlist for Sold-Out Event | A goer finds an event at full capacity and wants to join the waitlist in case a seat opens. |
| GO-04 | Use Map for Directions | A goer on the way to an event wants to tap the address and open directions quickly. |
| GO-05 | Share Event with Friends | A goer wants to send an event link to friends so they can plan attendance together. |
| GO-06 | Save Event Draft | A goer creating a personal event wants to save a draft first and publish later. |
| GO-07 | Limit Event Capacity | A goer organizing a meetup wants to set a participant cap to avoid overcrowding. |
| GO-08 | Broadcast Update to Attendees | A goer hosting an event wants to send one message to all joined participants about updates. |
| GO-09 | Set Recurring Schedule | A goer running a weekly club wants to repeat one event template instead of creating new entries every time. |
| GO-10 | Review Event Performance | After hosting an event, a goer wants to compare views and joins to understand interest. |

## Business Scenarios

| ID | Scenario Name | Scenario Description |
| :--- | :--- | :--- |
| BIZ-01 | Promote Event Placement | A business launching a campaign wants paid placement so its event appears higher in discovery results. |
| BIZ-02 | Build Branded Profile | A business wants a branded profile with website and social links to strengthen audience trust. |
| BIZ-03 | Capture Leads from Attendees | A business host wants to offer coupons or downloadable resources to convert attendees into leads. |
| BIZ-04 | Co-Host with Partners | A business wants to tag partner hosts and run co-branded events for shared visibility. |
| BIZ-05 | Measure Ad Conversion ROI | A business wants to compare ad spend and ticket joins to optimize marketing budget decisions. |
| BIZ-06 | Save Event Draft | A business event manager wants to save an event as draft while final details are being approved. |
| BIZ-07 | Control Event Capacity | A business organizer wants to define a maximum attendee limit to keep operations safe and manageable. |
| BIZ-08 | Send Bulk Attendee Notice | A business organizer wants to broadcast one message to all joined goers about last-minute updates. |
| BIZ-09 | Run Recurring Campaign Event | A business running monthly sessions wants recurring event setup to reduce manual work. |
| BIZ-10 | Analyze Post-Event Results | A business organizer wants to review view-to-join results to evaluate campaign performance. |

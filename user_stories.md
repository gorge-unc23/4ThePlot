# EventApp: Project Requirements and User Stories

## 1. The Organizer Role
| ID | Title | User Story |
| :--- | :--- | :--- |
| ORG-01 | Detailed Event Drafting | As an Organizer, I want to save an event as a draft so that I can refine the details before making it visible to the public. |
| ORG-02 | Ticketing and Capacity Limits | As an Organizer, I want to set a maximum attendee limit so that the event does not become overcrowded. |
| ORG-03 | Direct Attendee Communication | As an Organizer, I want to send a broadcast message to all joined Goers so that I can provide or special instructions. |
| ORG-04 | Recurring Event Logic | As an Organizer, I want to set an event to repeat weekly or monthly so that I do not have to manually recreate the listing. |
| ORG-05 | Post-Event Analytics | As an Organizer, I want to see a summary of how many people viewed versus joined my event so that I can measure the effectiveness of my promotion. |

---

## 2. The Goer Role
| ID | Title | User Story |
| :--- | :--- | :--- |
| GO-01 | Personalized Discovery | As a Goer, I want to follow specific event categories that match my personal hobbies. |
| GO-02 | Social Proof and Reviews | As a Goer, I want to read reviews and see ratings of an Organizer before I commit. |
| GO-03 | Waitlist Management | As a Goer, I want to join a waitlist for a sold out event so that I am notified if someone else cancels. |
| GO-04 | In-App Map Navigation | As a Goer, I want to tap the event address to open a map so that I can get directions. |
| GO-05 | Friend Invitations | As a Goer, I want to share a link or invite a contact to an event so that I can coordinate plans with my friends within the platform. |

---

## 3. The Admin Role
| ID | Title | User Story |
| :--- | :--- | :--- |
| ADM-01 | Report Processing | As an Admin, I want to access a prioritized queue of reported content so that I can quickly address harassment or illegal event listings to keep users safe. |
| ADM-02 | Verification Badge Issuance | As an Admin, I want to assign verified status to trusted professional organizers so that users know which events are officially sanctioned and low-risk. |
| ADM-03 | Global Banner Announcements | As an Admin, I want to push a system-wide banner notification so that I can inform all users about scheduled maintenance or major platform updates. |
| ADM-04 | Dispute Resolution | As an Admin, I want to access the chat logs between an Organizer and a Goer so that I can fairly resolve conflicts or refund requests based on evidence. |
| ADM-05 | Revenue and Growth Monitoring | As an Admin, I want to view a dashboard of daily active users and new event creations so that I can track the platform's growth and identify marketing needs. |

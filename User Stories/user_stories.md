# 4ThePlot: Frontend-Aligned User Stories

## 1. Common Stories

| ID | Title | User Story |
| :--- | :--- | :--- |
| UC-COM-01 | Register Account | As a Visitor, I want to create a Goer or Business account from the signup flow so that I can enter the role-specific app experience. |
| UC-COM-02 | Log In | As a Registered User, I want to log in with email and password so that I can open my role-specific home screen. |
| UC-COM-03 | View Landing Page Entry Actions | As a Visitor, I want to choose Login or Sign up from the landing page so that I can start the correct entry flow. |
| UC-COM-04 | Edit Profile | As a Logged-in User, I want to update my visible profile information from Settings so that my account details stay current. |
| UC-COM-05 | Change Appearance Settings | As a Logged-in User, I want to change the app appearance from Settings so that the interface matches my preference. |
| UC-COM-06 | View Global Notifications | As a Goer or Business User, I want to read platform announcements from Settings so that I can see current app messages. |

---

## 2. Goer Stories

| ID | Title | User Story |
| :--- | :--- | :--- |
| UC-GO-01 | Discover Trending Events | As a Goer, I want to browse trending event cards so that I can quickly find popular events. |
| UC-GO-02 | View Nearby Events | As a Goer, I want to view events near my device location so that I can discover accessible events around me. |
| UC-GO-03 | Search Events | As a Goer, I want to search events by text so that I can find events, cities, or venues faster. |
| UC-GO-04 | View Event Details | As a Goer, I want to open an event details page so that I can review the image, host, location, capacity, tags, comments, and actions. |
| UC-GO-05 | View Host Profile | As a Goer, I want to open the host profile from an event so that I can inspect the host and their hosted events. |
| UC-GO-06 | Join Event | As a Goer, I want to join an event from the event details flow so that the event appears in my joined events. |
| UC-GO-07 | Unregister From Event | As a Goer, I want to unregister from a joined event so that my attendance is removed. |
| UC-GO-08 | View Joined Events Calendar | As a Goer, I want to view joined events on the calendar so that I can organize events by date. |
| UC-GO-09 | View Events on Map | As a Goer, I want to view event locations on the map so that I can inspect events geographically. |
| UC-GO-10 | Add Event Comment | As a Goer, I want to add a comment on an event so that I can participate in the event discussion. |
| UC-GO-11 | Share Event | As a Goer, I want to share an event from the event details page so that I can send the event outside the app. |
| UC-GO-12 | Report Event | As a Goer, I want to report an event from its details page so that unsafe or inappropriate content can be reviewed. |
| UC-GO-13 | Report Comment | As a Goer, I want to report a comment so that harmful discussion content can be reviewed. |
| UC-GO-14 | Report Host | As a Goer, I want to report a host profile so that suspicious host behavior can be reviewed. |
| UC-GO-15 | Submit Host Verification Request | As a Goer, I want to submit a host verification request with documents so that I can ask to become a trusted host. |
| UC-GO-16 | Delete Host Verification Request | As a Goer, I want to delete my host verification request so that I can remove a pending submission. |

## 3. Business Stories

| ID | Title | User Story |
| :--- | :--- | :--- |
| UC-BIZ-01 | Create Business Account | As a Business User, I want to create a business account with role-specific profile information so that I can manage hosted events. |
| UC-BIZ-02 | View Hosted Events Tab | As a Business User, I want to open the Hosted tab so that I can see events created by my account. |
| UC-BIZ-03 | Create Hosted Event | As a Business User, I want to create a hosted event through the add event flow so that it becomes visible in the app. |
| UC-BIZ-04 | Upload Event or Profile Images | As a Business User, I want to upload event and profile images through visible pickers so that my pages show the correct media. |
| UC-BIZ-05 | Edit Hosted Event | As a Business User, I want to edit a hosted event so that I can update details, images, location, and capacity. |
| UC-BIZ-06 | Delete Hosted Event | As a Business User, I want to delete a hosted event after confirmation so that it is removed from visible event lists. |
| UC-BIZ-07 | View Hosted Events Calendar | As a Business User, I want to view my hosted events on the calendar so that I can review them by date. |
| UC-BIZ-08 | Filter Hosted Events by City | As a Business User, I want to filter hosted calendar events by city so that I can focus on one location. |
| UC-BIZ-09 | Promote Hosted Event as Trending | As a Business User, I want to mark a hosted event as trending so that it appears in the trending event area. |
| UC-BIZ-10 | View Business Profile and Hosted Event List | As a Business User, I want my business profile and hosted event list to be visible so that users can inspect my hosted events. |

## 4. Admin Stories

| ID | Title | User Story |
| :--- | :--- | :--- |
| UC-ADM-01 | View Admin Dashboard | As an Admin, I want to open the dashboard so that I can see platform summary cards and navigate to admin tools. |
| UC-ADM-02 | View and Manage Events | As an Admin, I want to view all events so that I can inspect platform event content. |
| UC-ADM-03 | Delete Event | As an Admin, I want to delete an event from the admin UI so that inappropriate or invalid events are removed. |
| UC-ADM-04 | Review Safety Reports | As an Admin, I want to filter and open safety reports so that I can review reported content. |
| UC-ADM-05 | Apply Moderation Action | As an Admin, I want to apply a moderation action with a reason so that the report records the action taken. |
| UC-ADM-06 | Update Report Status | As an Admin, I want to update a report status so that the visible report workflow stays current. |
| UC-ADM-07 | Review Host Verification Requests | As an Admin, I want to view host verification requests by status so that I can choose requests to inspect. |
| UC-ADM-08 | Review Host Verification Details | As an Admin, I want to open verification details so that I can inspect host information, documents, status, and history. |
| UC-ADM-09 | Approve or Reject Host Verification | As an Admin, I want to approve, reject, mark pending, or mark fraud with a reason so that host trust state is updated. |
| UC-ADM-10 | Publish or Edit Global Notification | As an Admin, I want to create or edit global notifications so that users can see platform announcements. |
| UC-ADM-11 | Monitor Platform Metrics | As an Admin, I want to review platform metrics and daily charts so that I can understand visible growth indicators. |
| UC-ADM-12 | Review Disputes | As an Admin, I want to view and filter disputes so that I can open cases that need review. |
| UC-ADM-13 | View Dispute Chat Logs | As an Admin, I want to view dispute chat logs so that I can inspect the evidence shown for a dispute. |
| UC-ADM-14 | Resolve Dispute | As an Admin, I want to submit a dispute decision and reason so that the case status is updated. |
| UC-ADM-15 | View Audit Logs | As an Admin, I want to page through audit logs so that I can review recorded platform actions. |
| UC-ADM-16 | Navigate Admin Tabs | As an Admin, I want to switch between Admin, Events, Map, and Profile tabs so that I can move through admin screens. |
